README3.txt - Unit Test Instructions

To run the unit tests for the arithmetic functions:

1. Ensure you have Python 3.x installed
2. Open a terminal/command prompt
3. Navigate to the directory containing this file
4. Run one of these commands:

   a) Run tests directly:
   python test_functions.py

   b) Run using unittest module:
   python -m unittest test_functions.py -v

Expected Output:
You should see test results showing 'OK' if all tests pass,
or detailed error messages if any tests fail.

Test Cases Included:
- Addition
- Subtraction
- Multiplication
- Division (including zero division check)
