README1.txt - Run Code Instructions

Install Nextjs: type "npm install next" into the terminal
Install Tailwind: type "npm install tailwindcss @tailwindcss/postcss postcss" into the terminal
Install Clerk: type "npm install @clerk/nextjs" , "npm install @clerk/clerk-react" into the terminal
Install Shadcn: type "npx shadcn@latest init", "npx shadcn@latest add input", "npx shadcn@latest add textarea" into the terminal
Type "npm run dev" into the terminal to run the site
