The codefiles ListTableCreate and UserTableCreate are query codes that, when executed, create and formate the tables that will store our user data.
The codefiles ListTableInsert and UserTableInsert are query codes that, when executed, read the .txt files for data and insert them into the proper formated tables.
The ListTableInsert and UserTableInsert can use ListingTest.txt and UserTest.txt, respectivaly, to created test table. (the path url needs to be updated to the url location of said .txt files)
These query files run automaticly through programs called agents that update periodicly, on server activation.
